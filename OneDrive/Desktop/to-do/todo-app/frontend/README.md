# Todo Frontend

This is the React frontend for the Todo application.

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn

## Installation

1. Navigate to the frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

## Running the Application

To start the development server:
```bash
npm start
```

The application will be available at `http://localhost:3000`.

## Environment Variables

- `REACT_APP_API_BASE_URL` - Base URL for the backend API (defaults to `http://localhost:8080/api/todos`)

## Scripts

- `npm start` - Start the development server
- `npm run build` - Build the production-ready application
- `npm test` - Run tests
- `npm run eject` - Eject from Create React App (permanent action)
